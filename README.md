# netMonitor

![Python 3+](https://img.shields.io/badge/Python-3+-brightgreen.svg?style=flat)

## 功能

双子楼校园网络状态不稳定，该工具可实现自动检测网络状态，重新登录保证网络状态。

## 用法

python3 netMonitor.py -u 校园网账号 -p 密码 -l False

## 原理

每隔一秒刷新一次百度主页，timeout设置为0.5，超时重新进行校园网登录认证。

## 环境要求

python3以上
